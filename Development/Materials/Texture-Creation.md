# Texture Creation

Textures may be easily created using the new vtex2 command line and GUI tool.

## Compressed Image Formats

As of August 2022, Chaos Engine has native support for the following compressed texture formats:
* DXT1
* DXT5
* BC4 (aka ATI1N)
* BC5 (via ATI2N)
* BC7 (VTF 7.6 Only)
